(async) executeQuery(intentData, dateRangeopt) → {Promise.<Object>}

Servicio que traduce intenciones interpretadas en consultas SQL reales. Soporta diferentes tipos de respuestas: resúmenes (KPI), tablas y texto. Organizado por categorías de datos: empleados, productos, clientes y ventas.
Parameters:
Name Type Attributes Description
intentData Object Datos de intención interpretados por OpenAI
Properties
Name Type Attributes Description
intent string Tipo de consulta a ejecutar
parameters Object <optional>
 Parámetros adicionales para la consulta
dateRange Object <optional>
 Rango de fechas para consultas temporales
Properties
Name Type Description
from string Fecha de inicio (YYYY-MM-DD)
to string Fecha de fin (YYYY-MM-DD)
Source:
  • services/queryService.js, line 3
Throws:
"dateRange is required for [intent]" - Cuando una consulta necesita rango de fechas
Type
Error
Returns:
Resultado estructurado de la consulta
Type
Promise.<Object>
Examples
// Consulta de total de empleados (KPI)
const result = await executeQuery({ intent: "employee_count" });
// Retorna: { type: "summary", content: "...", data: { total: 45 } }
// Consulta de ventas con rango de fechas
const result = await executeQuery(
  { intent: "sales_total" },
  { from: "2024-01-01", to: "2024-01-31" }
);
// Retorna: { type: "summary", content: "...", data: { total: 12500.50 } }
// Consulta de listado de productos (tabla)
const result = await executeQuery({ intent: "product_list" });
// Retorna: { type: "table", content: "...", data: { headers: [...], rows: [...] } }

(async) interpretQuestion(question) → {Promise.<Object>}

Utiliza el modelo GPT-4o-mini de OpenAI para interpretar preguntas en lenguaje natural y convertirlas en una estructura JSON que puede ser procesada por el sistema de consultas. Incluye el contexto del esquema de la base de datos para mejorar la precisión.
Parameters:
Name Type Description
question string Pregunta/consulta en lenguaje natural del usuario

    Type
    SyntaxError
  • Error de conexión con la API de OpenAI
    Type
    Error
Returns:
Objeto con la interpretación estructurada de la consulta
Type
Promise.<Object>
Examples
// Entrada: pregunta en lenguaje natural
const resultado = await interpretQuestion("¿Cuántas ventas hubo en enero?");

// Salida: estructura interpretada
{
  "intent": "sales_report",
  "needsDateRange": true,
  "parameters": {
    "metric": "count",
    "timeframe": "monthly",
    "month": "january"
  },
  "confidence": 0.92
}
// Otra posible salida:

  "needsDateRange": false,
  "parameters": {
    "category": "electronics",
    "sortBy": "price",
    "order": "desc"
  },
  "confidence": 0.87
}

(async) sendMessage(req, res) → {Promise.<Object>}

Procesa consultas en lenguaje natural, interpreta la intención mediante IA, y ejecuta consultas a la base de datos. Soporta filtrado por rango de fechas y control de acceso por roles.
