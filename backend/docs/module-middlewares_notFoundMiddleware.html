Module: middlewares/notFoundMiddleware

Middleware que responde con error 404 cuando ninguna ruta coincide con la solicitud. Debe ser el último middleware en la cadena de Express, después de todas las rutas definidas. Retorna una respuesta JSON con mensaje descriptivo y una imagen ilustrativa.
Source:
  • middlewares/error404.js, line 1
Examples
// Configuración típica en app.js:
const express = require('express');
const app = express();

// ... todas tus rutas aquí ...

// Middleware 404 - debe ir DESPUÉS de todas las rutas
const manage404 = require('./middlewares/notFoundMiddleware');
app.use(manage404);

// Middleware de error - debe ir DESPUÉS del 404
app.use((err, req, res, next) => {
  // manejo de errores
});
// Respuesta generada por este middleware:
// HTTP Status: 404 Not Found
// Content-Type: application/json
// Body:
{
  "msj": "404 not found",
  "img": "https://seranking.com/blog/wp-content/uploads/2021/01/404_01-min.jpg"
}
