Module: models/hrModel

Modelo que maneja todas las operaciones CRUD relacionadas con empleados. Incluye búsqueda por ID, nombre, y operaciones completas de gestión de personal. Normaliza datos (minúsculas) para consistencia en la base de datos.

(async, static) getEmployeeByName(first_name) → {Promise.<Array.<Object>>}

Realiza búsqueda case-insensitive (convierte a minúsculas). Puede retornar múltiples empleados si hay coincidencias parciales.

// Retorna: [
